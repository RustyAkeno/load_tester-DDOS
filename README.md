# load_tester-DDOS
Небольшой скрипт на пайтон для нагрузочного тестирования

Python-скрипт для нагрузочного тестирования сайта, который отправляет большое количество запросов с возможностью настройки количества запросов, их параллельности, HTTP-метода, данных и заголовков

Как пользоваться параметрами:

•  --min-wait (float): Минимальное время в секундах, которое каждый поток будет ждать между запросами. По умолчанию 0.0 (нет задержки).
•  --max-wait (float): Максимальное время в секундах, которое каждый поток будет ждать между запросами. По умолчанию 0.0 (нет задержки).

Примеры использования:

•  Пауза от 0.5 до 2 секунд между запросами для каждого "пользователя"

• python load_tester.py -u http://example.com -n 100 -c 10 --min-wait 0.5 --max-wait 2.0
*  Фиксированная пауза в 1 секунду (устанавливаем min и max одинаково):

• python load_tester.py -u http://example.com -n 50 -c 5 --min-wait 1.0 --max-wait 1.0
*  Отключение пауз (поведение по умолчанию):

• python load_tester.py -u http://example.com -n 100 -c 10
# Или явно:
    python load_tester.py -u http://example.com -n 100 -c 10 --min-wait 0.0 --max-wait 0.0
